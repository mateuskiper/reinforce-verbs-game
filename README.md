# reinforce-verbs-game
A pocket-sized game that helps Portuguese speakers master everyday English verbs.
It covers the 20 most frequent verbs in all forms (present, past, participle, future, and spoken contractions like *gonna*/*wanna*), the 9 modals, the auxiliary verbs *be*/*have*/*do*, and the *have*/*has*/*had* rule.
Every exercise follows a question-to-written-answer format: you type the correct form, which forces active production—a method shown by memory science to be more effective than mere recognition.
